オカダ・カズチカとは？ / ABOUT

新日本プロレスに「金の雨を降らせる男」として現れ、瞬く間に業界の頂点に駆け上がった現代プロレス界の象徴。その圧倒的な身体能力とスター性、そして勝負所で繰り出される必殺技「レインメーカー」で数々の伝説を築いてきた。リング内外で見せる王者の風格は、多くのファンを魅了し続けている。
